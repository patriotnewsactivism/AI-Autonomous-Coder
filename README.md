## 🆘 Troubleshooting

### "Voice recognition not supported"
- **Solution**: Use Chrome, Safari, or Edge on mobile
- **Alternative**: Type your message instead

### "Please enter an API key"
- **Solution**: Go to Settings and add your API key
- **Tip**: Use a FREE provider (Gemini, HuggingFace, Cohere)

### "API call failed"
- **Check**: Is your API key valid?
- **Check**: Do you have internet connection?
- **Check**: Have you exceeded daily limits?
- **Solution**: Try switching to a different FREE provider

### Response is slow
- **Try**: Switch to Gemini Flash (fastest)
- **Try**: Use simpler queries
- **Check**: Your internet speed
